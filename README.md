# Interactive Calculator

Clean calculator with basic operations, decimal support, and iOS-inspired design.

## 🚀 Live Demo
**[View Live App →](https://ayush-kaizen.github.io/javascript-calculator/)**

## ✨ Features
- ➕ Basic operations (add, subtract, multiply, divide)
- 🔢 Decimal point support
- 🔄 Chained calculations
- ⚠️ Error handling (division by zero)
- 🎨 iOS-inspired design
- 📱 Responsive layout

## 🛠️ Tech Stack
- JavaScript (functions, event listeners)
- CSS Grid for button layout
- DOM manipulation
